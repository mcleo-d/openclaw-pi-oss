# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

All participants in this project — including contributors, maintainers, and anyone engaging
via issues or pull requests — are expected to uphold these standards.

## Reporting

Instances of unacceptable behaviour may be reported privately to the project maintainer,
James McLeod:

- **Email:** <contact_mcleod@hotmail.co.uk> — recommended for sensitive reports
- **GitHub:** [@mcleo-d](https://github.com/mcleo-d) — for non-sensitive reports only

All reports will be treated with discretion. You can expect:

- Acknowledgement within **3 business days**
- A response addressing the report within **7–14 days**

If a report involves the maintainer, please reach out to a trusted independent party to
facilitate the process — this will be arranged on a case-by-case basis.

All reports are handled in accordance with the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
