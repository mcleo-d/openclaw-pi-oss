# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

All participants in this project — including contributors, maintainers, and anyone engaging
via issues or pull requests — are expected to uphold these standards.

## Reporting

Instances of unacceptable behaviour may be reported privately to the project maintainer,
James McLeod, via GitHub: [@mcleo-d](https://github.com/mcleo-d).

All reports will be treated with discretion and reviewed promptly.
